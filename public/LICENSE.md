## Auto-generated & external content

### GitHub stats cards

Files in `assets/github/` (e.g. `stats.svg`, `top-langs.svg`) are **automatically generated** by a GitHub Actions workflow (`.github/workflows/update-github-stats.yml`) using [github-readme-stats](https://github.com/anuraghazra/github-readme-stats). They reflect public GitHub activity data and are regenerated on a schedule.

These widgets are not original artwork. They may be referenced when forking this README structure, but their underlying data and rendering belong to the respective services.

### External embeds

Some README elements load from third-party URLs (e.g. typing animations, shields.io badges). Those services have their own terms and licenses. Mark Siazon does not control or redistribute that content.
